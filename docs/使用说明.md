# VoiceInput 使用说明

> macOS 语音输入工具 —— 按一下快捷键，说话，文字自动上屏。

---

## 一句话介绍

**VoiceInput** 是一个常驻菜单栏的 macOS 小工具。按下 `Option + 1` 开始录音，再按一次结束，系统自动把你说的话转成文字、整理好、直接输入到你当前光标所在位置。

---

## 安装

### 方式一：直接运行 App

1. 打开 `build/VoiceInput.app`
2. macOS 可能会提示"无法验证开发者"，右键（或按住 Control 点击）选择**打开**即可

### 方式二：自己编译

```bash
cd /path/to/type
./scripts/build_app.sh
open build/VoiceInput.app
```

---

## 首次使用：授予权限

第一次启动时，macOS 会弹出权限请求，需要允许以下权限：

| 权限 | 用途 | 必须 |
|------|------|------|
| **麦克风权限** | 录制你的声音 | ✅ 必须 |
| **辅助功能权限** | 自动粘贴文字到当前输入框 | ✅ 建议开启 |

如果首次启动时拒绝了权限，可以随时在 **系统设置 > 隐私与安全性** 中开启。

> 💡 如果 `Option + 1` 无法全局触发，可能还需要在 **系统设置 > 隐私与安全性 > 输入监控** 中添加 VoiceInput。

---

## 快速上手

### 第一步：配置 API Key

1. 点击菜单栏的 🎤 麦克风图标
2. 选择 **设置...**
3. 填入你的**硅基流动 API Key**（可在 [siliconflow.cn](https://siliconflow.cn) 获取）
4. 点击**保存**

### 第二步：开始使用

```
1. 把光标放在你想输入文字的地方（微信、备忘录、浏览器等任意输入框）
2. 按 Option + 1 —— 屏幕底部出现"正在听"悬浮条
3. 开始说话
4. 说完再按一次 Option + 1 —— 等待"整理中"完成
5. 文字自动粘贴到你的输入框 ✅
```

就是这么简单。

## 界面说明

### 菜单栏图标 🎤

常驻在屏幕右上角菜单栏，点击后可操作：

- **开始录音 / 停止录音** — 同快捷键功能
- **复制最近结果** — 手动复制上一次语音输入的内容
- **设置...** — 打开配置页面
- **权限状态** — 查看麦克风和辅助功能权限是否已授权
- **退出** — 关闭应用

### 悬浮状态条

录音和处理期间，屏幕底部会显示一个小型胶囊状态条：

| 状态 | 显示 | 说明 |
|------|------|------|
| 🎤 正在听 | 录音中，带有声波动画 | 继续说话 |
| ⏳ 整理中 | 正在转写和 AI 整理 | 请稍候 |
| ✅ 已输入 | 完成，文字已自动粘贴 | 可以继续了 |
| ❌ 处理失败 | 出错，可能已复制原始转写 | 查看剪贴板 |

## 设置项详解

在菜单栏打开 **设置...**，可配置以下项目：

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| **Base URL** | API 地址 | `https://api.siliconflow.cn/v1` |
| **API Key** | 硅基流动的 API 密钥 | 空，需填写 |
| **语音转文字模型** | STT 模型 | `FunAudioLLM/SenseVoiceSmall` |
| **文本整理模型** | AI 整理模型 | `Pro/zai-org/GLM-5.1` |
| **API 超时** | 接口超时时间 | 45 秒 |
| **开机自启** | 登录 Mac 时自动启动 | 关闭 |

> 你可以在设置页点击"测试文本整理"按钮，快速验证配置是否可用。

## 整理效果

系统会对你的口语做以下处理：

- ✅ 去除"呃、嗯、然后就是、那个"等口头禅
- ✅ 自动添加中文标点（逗号、句号、问号等）
- ✅ 根据语义自然分段
- ✅ 去掉无意义的重复语句
- ✅ 轻微调整语序使表达更顺畅
- ❌ **不改变你的原意**
- ❌ **不添加你没有说过的新内容**
- ❌ **不擅自扩写你的观点**

---

## 意外情况处理

| 情况 | 会发生什么 |
|------|-----------|
| 没有输入框/无法粘贴 | 结果自动复制到剪贴板，手动粘贴即可 |
| 辅助功能权限未开启 | 结果复制到剪贴板，不会自动粘贴 |
| 语音转文字失败 | 不粘贴，显示"处理失败"错误提示 |
| 转写成功但 AI 整理失败 | **复制原始转写文字**到剪贴板，不会丢内容 |
| 网络超时 | 显示错误，录音文件临时保留，可再次尝试 |

## 常见问题

**Q: 快捷键和其他 App 冲突怎么办？**
A: 目前快捷键固定为 `Option + 1`，如果与其他应用冲突，可先在那些应用中修改快捷键设置。

**Q: 录音最长多久？**
A: 实践中建议录制 30 秒到 3 分钟的语音片段。API 限制音频不超过 1 小时、50MB。

**Q: API Key 安全吗？**
A: API Key 存储在 macOS Keychain（系统钥匙串）中，不会被写入日志或代码仓库。

**Q: 录音文件会保存吗？**
A: 不会。处理完成后临时录音文件自动删除，本地只保留最近 10 条文字历史记录。

**Q: 语音数据会上传到哪？**
A: 音频仅上传到你配置的 API 地址（默认硅基流动）进行语音转文字处理。

**Q: 支持英文吗？**
A: 支持中英混合输入，但优先优化了中文体验。
