# 更新日志

本项目遵循语义化版本。日期为本地时区。

## v0.2.0 — 2026-07-01

### 新增
- 文本润色支持多服务商：硅基流动 / DeepSeek / 阿里百炼，语音转文字与文本润色可各自独立配置（Base URL、模型、按服务商保存的 API Key）。默认文本服务商为 DeepSeek（`deepseek-v4-flash`）。
- 短句直发：转写文本短于设定字数（默认 8，可调）时跳过润色、直接发送，减少一次网络往返。
- 录音或处理过程中按 Esc 取消，丢弃当前录音、并中断正在进行的请求（Esc 仅在录音/处理时临时生效）。
- 深色玻璃悬浮状态窗（真实通透材质、圆角、自然双层阴影）。
- 应用图标（深色波形）。
- 粘贴前判断当前焦点是否可编辑，并在关闭“始终复制到剪贴板”时于粘贴后恢复原剪贴板内容。

### 变更
- 整理输出改为纯文本，保持用户第一人称口吻，不添加评论或助手式开场，不使用 Markdown（分点用 1、2、3）。
- 语音转文字使用独立且更长的超时（至少 300 秒），适应较长录音；文本润色仍用设置中的超时值。

### 修复
- 修复打开设置页“打开→关闭→再打开”时的闪退（NSWindow 释放后被重复引用导致 EXC_BAD_ACCESS）。
- 失败提示在悬浮窗停留更久（6 秒）以便读取，失败原因写入系统日志（subsystem `cn.local.voiceinput`）。
- 提升自动粘贴可靠性：粘贴前等待 Option 等修饰键释放、并给剪贴板写入留出时间，避免合成成 Cmd+Option+V 或粘到空内容。

## v0.1.0

- 首个 MVP：Option + 1 语音输入，硅基流动 STT + 文本整理，自动粘贴与剪贴板兜底，菜单栏常驻，历史记录，Keychain 存 API Key，开机自启，权限状态显示。
